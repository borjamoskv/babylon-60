Security

Notas de postura de seguridad pensadas para entornos productivos y auditoría.

Tenant scope

  • Aislamiento por tenant en lecturas y escrituras.
  • Validación de firmas/hash en eventos persistidos.

Protección de datos

  • Criptografía en reposo y detección de secretos en texto.
  • Reglas de redacción y red flags para secretos.

Auditoría

  • Cadena de hash y evidencias de continuidad.
  • Verificación periódica del estado de integridad.
