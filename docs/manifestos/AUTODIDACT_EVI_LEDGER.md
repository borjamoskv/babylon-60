## 1. Extracción Isomórfica (Desmitificación)
La literatura sobre Expected Value of Information (EVI), Surprise Value of Information (SVI), Brier Score y Mercados de Predicción se consolida en una única topología estructural en CORTEX-Persist. El "Hypothesis Ledger" no es un log narrativo, sino una tabla SQL estricta donde cada agente colapsa su incertidumbre en una probabilidad antes del testeo, siendo castigado termodinámicamente si falla la predicción (alto Brier Score). El SVI dicta qué ramas merecen ser exploradas basándose en el ratio Coste/EVI (CBR) y el ROI_surprise.

## 1.5 Las 10 Primitivas de Máxima Exergía para la Mitigación / Ejecución
- **EVI-001**: `Expected Value of Information (EVI)` - Cuantifica la utilidad termodinámica de adquirir información antes de mutar el sistema.
- **EVI-002**: `Surprise Value of Information (SVI)` - Mide entropía de un evento para forzar asignación de atención (Attentional Routing).
- **EVI-003**: `Surprisal-Probability Index (SPI)` - Filtra ruido estadístico, ponderando SVI por relevancia o probabilidad conjunta.
- **EVI-004**: `Brier Score Calibration` - Métrica estrictamente propia que audita la certidumbre probabilística de los agentes.
- **EVI-005**: `Hypothesis Ledger` - Tabla inmutable donde las conjeturas de los LLMs se registran como apuestas probabilísticas.
- **EVI-006**: `Epistemic Discipline` - Obligación del enjambre de formalizar la incertidumbre; erradica aserciones gratuitas.
- **EVI-007**: `Cost-Benefit Ratio (CBR)` - Ratio entre el coste computacional y el EVI para guiar sondeos.
- **EVI-008**: `ROI_surprise Metric` - Valor asimétrico de investigar una rama mediante la fórmula (EVI * SVI*) / Cost.
- **EVI-009**: `Dynamic Confidence Threshold` - El umbral de admisión de un agente escala inversamente proporcional a su Brier Score.
- **EVI-010**: `SAGA-Abort Penalty` - Castigo en exergía cuando una mutación prometida explota en tiempo real.
