# Plataforma Claude - Índice de Referencia

**Estado C5-REAL:** Documentación extraída de la Plataforma Claude (Anthropic).

## Componentes Principales
1. **Mensajes:** Acceso directo al modelo. Gestión manual del turno, estado de la conversación y bucle de herramientas.
2. **Agentes Gestionados (Managed Agents):** Infraestructura de agentes totalmente gestionada. Despliegue y administración de agentes autónomos en sesiones con estado e historial de eventos persistente.
3. **Administración:** Gestión de claves API, monitorización de uso, migración de modelos y configuración de espacios de trabajo.

## Familia de Modelos Claude
- **Claude Fábula 5 (claude-fabula-5):** *El más capaz.* La máxima capacidad para el razonamiento más exigente y el trabajo agéntico de largo horizonte.
- **Claude Opus 4.8 (claude-opus-4-8):** *Avanzado.* Excelente para análisis complejos, programación y tareas creativas que requieren razonamiento profundo.
- **Claude Soneto 4.6 (claude-sonnet-4-6):** *Mejor equilibrio.* Equilibrio ideal entre inteligencia y velocidad para la mayoría de las cargas de trabajo en producción.
- **Claude Haiku 4.5 (claude-haiku-4-5):** *El más rápido.* Respuestas ultrarrápidas para aplicaciones de alto volumen y sensibles a la latencia.

## Superficies y SDKs
- **Lenguajes Soportados:** Python, TypeScript, Go, Java, Ruby, PHP, C#.
- **Integraciones de Infraestructura:** Amazon Bedrock, Google Cloud Vertex AI, Microsoft Foundry.
- **Interfaces Directas:** cURL, CLI.
