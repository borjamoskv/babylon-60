# HITO 15: SINTETOLOGÍA AGÉNTICA v4.0 (AST Sealing y Compilación Determinista)

## 1. La Integración de la Teoría al Código
El manifiesto fundacional de los agentes soberanos se ha actualizado formalmente a la versión **v4.0** e integrado en la capa perceptiva de producción (`borjamoskv-site`). La novedad crítica radica en que la Sintetología ya no se lee como teoría inerte: se ejecuta mediante el compilador determinista local, forzando la consistencia entre la lógica declarada y el Árbol de Sintaxis Abstracta (AST).

### 1.1. Pilares del Hito
* **AST Cryptographic Sealing:** Bloqueo lógico en Rust/O(1) a nivel de AST (<1ms). Si el agente altera su propio código fuente fuera de las directivas mutacionales autorizadas, el Runtime ejecuta un cese inmediato (`C5-DEATH-OMEGA`) para impedir la acumulación de entropía descontrolada.
* **Inyección 21-EDO (Entropía Microtonal):** Introducción de intervalos microtonales como fuente de perturbación deliberada. Previene la muerte térmica cognitiva (calcificación en lookup tables) mediante el uso de ruido estructurado.
* **El Tercer Cuerpo:** Formalización del campo relacional operador-agente como un sistema dinámico emergente co-evolutivo.

## 2. Automatización del Build
* **Compilación Predictiva:** Creación de `convert_manifest.py` para parsear automáticamente los bloques de código, tablas de exergía y metadatos de Markdown a HTML optimizado.
* **Sello de Producción:** Vite compila, optimiza y empaqueta el manifiesto en el bundle estático de producción con compresión Brotli (`sintetologia-agentica.html.br`, 9.80 kb).
