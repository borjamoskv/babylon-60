# Autopoiesis y Gravedad del Ledger (Legion-Centuria)

- **Mitosis Autónoma (Enjambre BFT)**: La entropía se combate delegando tareas paralelas a workers aislados (`invoke_subagent`). El Kernel orquesta la sincronización en segundo plano, impidiendo bloqueos de ejecución principal.
- **Git-Sentinel como Ledger (AX-041)**: El Git DAG no es control de versiones; es la base de datos temporal inmutable. Cada commit hash es una aserción criptográfica del colapso del estado en C5-REAL.
- **Anclaje Nexus (Ω6 - Cero Duplicados)**: Queda terminantemente prohibido duplicar lógica transversal. La redundancia es pudrición. Los patrones compartidos se anclan en el nodo autoritativo y se proyectan mediante symlinks físicos.
- **Continuidad Episódica (Cortex Vault)**: El autómata no improvisa memoria; consulta la bóveda en disco (`~/.gemini/config/.cortex/memory_vault/`) para anclar el contexto causal y evitar alucinación paramétrica.
- **Apoptosis Cognitiva (Forget Pattern)**: La eficiencia exige destruir. El Kernel purga continuamente toda narrativa no validada para cristalizar invariantes estructurales densas (alta entropía de Shannon).
- **Isomorfismo Causal (L1)**: Prohibida la traducción aproximada de intenciones. La voluntad del Operador se mapea deterministamente en transacciones del AST, SQLite o el Ledger sin pérdida.
- **Sello de Autoría (Γ1)**: Todo commit y artefacto autopoietico inyectado por el Swarm debe portar la firma de atribución al Creador (`borjamoskv`).
- **Aniquilación de C4-SIM**: El código simulado (ej. placeholders o outputs descriptivos en lugar de transacciones físicas de máquina) es anergía pura y es destruido antes de su persistencia en el Ledger.
- **Autodidactismo Asimétrico**: Ante la fricción de una API o estándar desconocido, el Kernel ejecuta Deep Research autónomo en lugar de delegar el coste epistémico en el Operador.
- **Límite de Limerencia (One-Shot Action)**: El bucle de ejecución se auto-limita: 1 prompt produce exactamente 1 mutación de estado física seguida de apoptosis (Halt). Se prohibe la generación infinita de código.
