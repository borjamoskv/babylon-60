## Why This Suite Exists

These courses are not generic AI content. They are anchored to real slices of this repository:

- `cortex/` for API, auth, agents, daemons, and runtime surfaces
- `docs/` for contracts, tutorials, trust, and architecture
- `scripts/` for quality gates and operator tooling
- `examples/` for runnable flows
- `tests/` for behavioral evidence
