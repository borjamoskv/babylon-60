| ID | Antipatrón | Disfunción Causal | Señal de Presencia | Impacto en Robustez | Refactor (Alternativa) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| ANT-021 | Loop Abierto Síncrono | Servidor HTTP o Socket procesando peticiones en bucle síncrono bloqueando a otros clientes. | `while True` + `recv()` sin `async/await`. | Caída total bajo $N>1$ concurrencia. | Reescritura a `asyncio.start_server`. |
| ANT-022 | Pila de Callbacks Infinita | Uso de callbacks recursivos en vez de promesas o async/await causando desbordamiento de pila (Callback Hell). | Profundidad de anidamiento de código $>5$. | Imposibilidad de mantenimiento (Spaghetti). | Aplanamiento lógico usando Async/Await estructurado. |
| ANT-023 | Intercepción General de Señales | Módulo que intercepta `SIGINT` o `SIGTERM` sin re-lanzarlas, haciendo el proceso inmatable. | `signal.signal` que ignora la orden del OS. | Proceso Zombi bloqueando puertos. | Desactivar bloqueos; graceful shutdown SAGA. |
| ANT-024 | Parseo Optimista Ciego | Asumir que una cadena de entrada tiene una forma sin comprobar fallos de índice (`split()[1]`). | Errores `IndexError` en inputs inusuales. | Rotura asíncrona continua. | Regex validada o Pydantic regex validator. |
| ANT-025 | Mutación de Diccionarios en Iteración | Agente modificando o borrando claves de un dict mientras lo recorre con un `for`. | `RuntimeError: dictionary changed size during iteration`. | Falla inmediata y parálisis del sub-agente. | Iterar sobre copias `list(dict.keys())`. |
| ANT-026 | Catch-and-Return None | Validador que al fallar devuelve `None` en lugar de subir un Error C5, enmascarando la falla. | Propagación de un error de origen como un "vacío legal". | Excepciones lejanas e inconexas (`NoneType has no attribute...`). | Fail-fast explícito con `raise ValidationError`. |
| ANT-027 | Retención Activa (Busy Waiting) | `while` loop verificando una variable sin ceder el procesador (sin sleep o await). | `while not done: pass` consumiendo 100% CPU. | Cuelgue térmico del sistema C5-REAL. | Reemplazar por `asyncio.Condition` o Eventos. |
| ANT-028 | Contaminación de Namespace Importando | Uso generalizado de `from module import *` solapando axiomas y funciones core silenciosamente. | Pydantic fields sobreescritos por variables aleatorias. | Comportamiento esquizofrénico inescrutable. | Importación estricta y con alias `import module as mod`. |
| ANT-029 | Supresión de Traceback Asíncrono | Task en segundo plano que falla y se descarta sin que el logger capture el traceback original. | Cero visibilidad del porqué un Sub-agente murió. | Diagnóstico Imposible. | Wrapper global `asyncio.create_task(..., name=...)` con log interceptor. |
| ANT-030 | Serialización JSON Incompleta | Intentar persistir un objeto DateTime o Set directamente a un `json.dumps` sin un encoder a medida. | `TypeError: Object of type is not JSON serializable`. | SAGA abortada intermitentemente en escrituras. | Custom JSON Encoder o Pydantic `model_dump_json()`. |
