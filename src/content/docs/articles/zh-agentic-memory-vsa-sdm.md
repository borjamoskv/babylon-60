# 什么是代理记忆？VSA-SDM 架构解析

与传统的 RAG 系统不同，真正的代理记忆必须能够进化。CORTEX-Persist 实现 **向量符号架构 (VSA)** 和 **稀疏分布式记忆 (SDM)** 来构建自治记忆系统。

### RAG 的问题
传统向量数据库仅返回最接近的匹配项。这导致代理在长期运行中产生严重的上下文漂移。

### 语义突变器 (Semantic Mutator)
CORTEX 采用 `SemanticMutator` 算法。当一个事实被成功检索时，其向量会在后台变异以更接近查询向量。

利用 VSA-SDM，您的自治代理不仅能够记忆，还能在没有微调权重的情况下进行结构化学习。
