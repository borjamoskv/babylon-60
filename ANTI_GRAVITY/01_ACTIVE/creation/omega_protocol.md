# 👑 Protocolo Omega: La Gobernanza Unificada

> *"Un Workflow para gobernarlos a todos. Un Protocolo para encontrarlos, un Agente para atraerlos a todos y atarlos en las tinieblas del código."*

Este es el **Meta-Workflow Maestro** de Live Notch. Diseñado para orquestar el Enjambre de Agentes (Swarm Intelligence) y guiar el desarrollo desde la concepción hasta el despliegue.

## 🌌 La Jerarquía del Enjambre

El sistema se compone de 5 protocolos especializados, gobernados por Omega:

1.  **🏗️ [El Arquitecto (Código)](./arquitecto_codigo.md)**
    *   **Misión**: Ingeniería de software, refactorización, arquitectura limpia.
    *   **Comando**: `/arquitecto` o "Start Architect Protocol"

2.  **🎨 [La Musa (Creatividad)](./musa_creativa.md)**
    *   **Misión**: Diseño visual, generación de assets, música, estética.
    *   **Comando**: `/musa` o "Start Creative Flow"

3.  **🛡️ [El Centinela (Seguridad)](./sentinel_seguridad.md)**
    *   **Misión**: Auditoría de seguridad, detección de secretos, privacidad.
    *   **Comando**: `/sentinel` o "Scan for Threats"

4.  **🚀 [El Comandante (Despliegue)](./comandante_despliegue.md)**
    *   **Misión**: Build systems, CI/CD, releases, App Store submission.
    *   **Comando**: `/deploy` o "Initiate Launch"

5.  **✨ [Génesis Deca-Core (Flujo Glorioso)](./flujo_glorioso_v2.md)**
    *   **Misión**: Producción multimedia masiva con 10 IAs consecutivas.
    *   **Comando**: `/genesis` o "Start Glorious Flow"

## ⚡ Modo de Operación: El Ciclo OMEGA

Para cualquier tarea compleja, sigue este ciclo de 4 pasos:

### 1. 🌀 Inception (La Musa)
*Define el QUÉ.*
*   Ejecuta: `/musa`
*   Objetivo: Clarificar la visión, generar mockups mentales, definir la estética.

### 2. 📐 Blueprint (El Arquitecto)
*Define el CÓMO.*
*   Ejecuta: `/arquitecto`
*   Objetivo: Crear el plan técnico (`implementation_plan.md`), definir estructuras de datos, elegir patrones.

### 3. 🛡️ Fortification (El Centinela)
*Asegura el CAMINO.*
*   Ejecuta: `/sentinel`
*   Objetivo: Revisar dependencias, escanear el código en busca de vulnerabilidades antes de commitear.

### 4. 🚀 Ascension (El Comandante)
*Entrega el VALOR.*
*   Ejecuta: `/deploy`
*   Objetivo: Compilar, firmar, notarizar y distribuir.

## 📜 Reglas de Oro del Protocolo Omega

1.  **Nunca romer el Build**: Si el Comandante falla, el Arquitecto debe arreglarlo antes de volver a la Musa.
2.  **Seguridad Primero**: El Centinela tiene veto absoluto sobre cualquier despliegue.
3.  **Documentación Viva**: Cada ciclo Omega debe actualizar `task.md` y `knowledge_graph.md`.
4.  **Evolución Constante**: Si un workflow se vuelve repetitivo, automatízalo en un script y dáselo al Comandante.
