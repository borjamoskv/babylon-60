// Core Mechanics
[ 01 ]

Governed Write Path

Every write passes guards, schema validation, and encryption before touching storage. If a proposal fails validation, the write aborts. No silent corruption.

[ 02 ]

Cryptographic Audit Trail

SHA-256 hash-chain ledger. Every action logged, chained, and tamper-evident. Export the full chain for external audit at any time.

[ 03 ]

Temporal Truth

Facts change, history remains. Temporal versioning with revocation-aware recall. Know what was true, when, and why it changed.
