# 💡 TIPS — Quick Patterns

## Parallel Tool Calls
Use when tasks are independent. Never serialize unnecessarily.

## Git Atomicity
One commit = one logical change. Never bundle unrelated changes.

## Test-First for Bugs
Write the failing test BEFORE fixing the bug.

## KI-First for Research
Check Knowledge Items BEFORE doing web search.

## Shannon Compression
Token budget is finite. Eliminate rhetoric. Data, not prose.

## Entropy Measurement
Measure before AND after every change. If entropy increases, revert.
