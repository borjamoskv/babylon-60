# Confluencia 2: El Arquitecto Físico MMX

**Objetivo:** Desde la concepción legal del terreno hasta la simulación termodinámica y logística pesada de materiales terrestres.

// turbo-all
1. Ejecutar `cadastral-perimeter-check` para localizar el terreno con vacíos legales o riesgo 0 de expropiación.
2. Iniciar `scrap-negotiator` para negociar autónomamente los precios de chatarra, neumático, acero y cristal necesarios.
3. Usar `heavy-geo-logistics` para simular la ruta de los camiones, esquivando barro y excedente de límite de peso.
4. Aplicar el modelo `earthship-thermo-calculator` para simular y calcular la inercia térmica de los muros principales de tierra y llantas.
5. Ejecutar `biosphere-terraformation-omega` para diseñar el ciclo cerrado biológico de plantas y agua interna.
