# ✨ FLUJO GLORIOSO — Multimedia Production

## Pipeline
1. **Input**: Content brief or raw material
2. **Script**: Generate narrative/script (LLM)
3. **Audio**: Generate/acquire audio (`Sonic-Foundry-Omega`)
4. **Visual**: Generate visuals (Veo, generate_image)
5. **Assembly**: Combine in video pipeline (Remotion/FFmpeg)
6. **Polish**: Color grade, effects, transitions
7. **Export**: Format-optimized output
8. **Distribute**: Upload to platforms

## Quality Gate
Every output passes `teodosi-omega` evaluation before distribution.

> **Skills:** `Sonic-Foundry-Omega`, `Aesthetic-Foundry-Omega`
