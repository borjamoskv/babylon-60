01 — the problem

Agents forget. That's the bug.

Session memory dies when your process restarts. Vector DBs remember tokens, not intent. Every team ends up re-implementing the same brittle stack: embeddings here, metadata there, a cron job to evict stale rows, a second one to re-index.

02 — the primitive

Three guarantees. Nothing else to think about.

01 / PERSISTENT

Survives restarts, deploys and model swaps.

Memories outlive your process, your Kubernetes pod, and the next GPT release. Zero migration when you change providers.

await mem.remember("...")
02 / QUERYABLE

Recall by meaning, time, agent or metadata.

Semantic search, temporal filters, per-agent namespaces and arbitrary tags — composed in a single call, not a pipeline.

mem.recall(query, since="7d")
03 / OBSERVABLE

Inspect, edit and replay every memory write.

Full audit log of everything your agent has ever remembered. Rewind to any point in time. Ship explainable agents.

mem.audit(agent="agent-42")
03 — how it works

Four primitives. One API.

→ 01 / WRITE

Emit an event

Any JSON-serialisable fact your agent observes. Tags and metadata are free-form.

mem.remember(...)
→ 02 / EMBED

Embed once, cache forever

Pluggable encoders. Identical inputs are deduped at ingest — no double billing.

openai · cohere · local
→ 03 / STORE

Tiered hot / warm / cold

Hot memories stay in RAM. Cold ones drop to object storage. Retrieval is transparent.

RAM · SSD · S3
→ 04 / RECALL

Deterministic & fast

p99 under 50ms globally. Results include provenance, tags and the original write event.

mem.recall(...)
04 — the sdk

Built for people who'd rather read code than docs.

from cortexpersist import Memory

mem = Memory("agent-42", api_key=os.environ["CORTEX_KEY"])

# 1 — write
mem.remember("User booked a flight to Lisbon on 2026-05-14",
             tags=["travel", "booking"])

# 2 — recall
hits = mem.recall("where is the user going next month?", top_k=3)
for h in hits:
    print(h.text, h.score, h.latency_ms)

# 3 — audit
for event in mem.audit(since="24h"):
    print(event.actor, event.op, event.payload)
import { Memory } from "cortexpersist";

const mem = new Memory("agent-42", { apiKey: process.env.CORTEX_KEY });

// 1 — write
await mem.remember("User booked a flight to Lisbon on 2026-05-14", {
  tags: ["travel", "booking"],
});

// 2 — recall
const hits = await mem.recall("where is the user going next month?", { topK: 3 });
hits.forEach(h => console.log(h.text, h.score, h.latencyMs));

// 3 — audit
for await (const ev of mem.audit({ since: "24h" })) {
  console.log(ev.actor, ev.op, ev.payload);
}
package main

import (
    "context"
    "fmt"
    "github.com/cortexpersist/cortex-go"
)

func main() {
    ctx := context.Background()
    mem := cortex.New("agent-42")

    // write
    mem.Remember(ctx, "User prefers window seats", cortex.Tags{"travel"})

    // recall
    hits, _ := mem.Recall(ctx, "seat preference?", 3)
    for _, h := range hits {
        fmt.Println(h.Text, h.LatencyMs)
    }
}
