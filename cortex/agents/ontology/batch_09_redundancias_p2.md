| RED-029 | Generación Dummy de Fallback | Proveer respuestas estructurales JSON vacías pero válidas si el modelo falla, para no romper la UI. | Pantallas blancas o crashes (TypeError) en el frontend. | Muestra de datos vacíos al Usuario | Validadores Pydantic pre-response |
| RED-030 | Bucle Ouroboros Condicionado | Todo subagente debe tener un Timer Condition de rescate. | Un subagente se pierde en razonamiento circular infinito y no responde al parent. | Tarea background (schedule) | API de Temporizador (TimerCondition) |
| RED-031 | Vector Clock Empotrable | Inyectar reloj lógico (Lamport timestamp) en cada mensaje del Swarm. | Desorden causal en mensajes asíncronos (A procesado antes que B). | Campos extra en Payload JSON | Lógica de ordenación en consumidor |
| RED-032 | Cgroups / ulimit Enforcement | Limitar por OS los procesos y memoria que un script auto-generado puede consumir. | Código "Lab" ejecuta un bucle infinito que consume toda la RAM (OOM del Host). | Dificultad de setup cruzado (Mac/Linux) | Sandbox local / Kernel limits |
| RED-033 | Rotación de Contexto (Context Window Sliding) | Al llenar la ventana de tokens, resumir la mitad más antigua y concatenarla con la mitad más nueva. | `MaxTokensExceeded` interrumpiendo un hilo cognitivo profundo. | Coste de llamada de resumen (LLM) | Middleware de Memoria episódica |
| RED-034 | Comprobación Isomórfica de Diffs | Medir si el AST original y el AST nuevo difieren en más de lo teóricamente posible (Tasa de mutación anómala). | LLM reescribe todo el archivo borrando métodos que no debía tocar. | Parseo doble (antes y después) | AST Diff Engine (Python `ast` / `radon`) |
