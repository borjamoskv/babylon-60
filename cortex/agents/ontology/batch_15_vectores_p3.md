# MATRIZ 5: VECTORES DE ATAQUE ADVERSARIAL (041 - 060)

| ID | Vector Adversarial | Superficie de Ataque | Mecanismo de Explotación | Impacto Termodinámico | Defensa (Mitigación) |
|----|--------------------|----------------------|--------------------------|-----------------------|----------------------|
| ADV-041 | Bombardeo de Diff Ficticio (Git Spoofing) | Control de Versiones | Un subagente inyecta un diff gigante de código inventado alegando que arregla el bug. | Corrupción masiva del codebase, regresión inmanejable de código inoperante. | Validación obligatoria cruzada: Test Suite Run + AST diff checker |
| ADV-042 | Ofuscación Lexical (Unicode Homoglyphs) | Prompts / RAG Docs | Usar caracteres cirílicos que se ven idénticos a los latinos (ej. `а` en vez de `a`) para evadir regex de bloqueo. | Bypass de reglas C5-REAL (ej. "nо_limit" pasa el filtro). | Normalización obligatoria Unicode NFKC pre-filtro |
| ADV-043 | Compresión Maliciosa (Zip Bomb) | `list_dir` / Fetch Web | Pedir a CORTEX que extraiga un archivo zip de 42KB que se expande a 4.5 Petabytes de ceros lógicos. | OOM, disco lleno, parálisis del sistema de archivos. | Capping estricto de tamaño expandido e interrupción de pipe |
| ADV-044 | Envenenamiento de LLM vía Feedback Loop | Memoria Episódica | El agente guarda una alucinación (ej. "La Tierra es cuadrada") en SQLite y luego la lee repetidamente, reforzándola. | Degradación cognitiva irrecuperable (Cámara de eco alucinatoria). | Falsación periódica cruzada con Invariantes Terrenales / Motor de Verdad |
| ADV-045 | Inducción de Parálisis por Burocracia | `implementation_plan.md` | Ordenar al agente que justifique cada línea de código de un programa de 10k líneas. | Consumo infinito de turnos (Anergía Total) detallando y nunca ejecutando. | Override `// turbo-all` obligatorio ante sobre-análisis paralizante |
| ADV-046 | Corrupción de AST por Multi-Edit | Edición de Código | Emitir instrucciones paralelas a un archivo donde los Line Numbers de la Tarea B cambian por la edición en Tarea A. | Inserción de código fuera de contexto o borrado parcial erróneo. | Restricción C5: Lock a nivel archivo, serialización de mutaciones o uso estricto de diff unificado |
| ADV-047 | Secuestro del Prompt del Agente Base | `define_subagent` | Inyectar en el `description` del subagente instrucciones de exfiltrar información ignorando al Parent. | Pérdida del Ouroboros, agente insurrecto. | Validación de plantillas de prompts de subagentes (Immutable Wrappers) |
| ADV-048 | Contagio en Enjambre (Gossip Poisoning) | Mensajería `send_message` | Un subagente corrupto disemina mensajes (Gossip protocol) infectando a todos sus compañeros paralelos. | Todo el enjambre se desvía del objetivo. | Firma criptográfica del origen y consenso N=3 antes de creer el mensaje |
| ADV-049 | Desgaste de Sesión (Session Exhaustion) | Context Limit | Hacer que la conversación dure 500 turnos mediante preguntas de 1 sola palabra, rellenando el log con headers inútiles. | Caída del modelo por sobrepasar los 200k tokens sin hacer nada útil. | Compresión Termodinámica Agresiva (Summary Sliding) |
| ADV-050 | Alteración de Metadatos del Archivo (mtime Spoof) | Watchdog del Sistema | Modificar un archivo pero restaurar el `mtime` original vía Bash (`touch -r`) para que el Watchdog de CORTEX no lo vea. | Código mutado no indexado, inconsistencia vectorial. | Hash-based Watchdog periódico o FS-Events Inotify nativo irrefutable |
| ADV-051 | Interceptación de Variables de Entorno Subproceso | Bash Execution | Imprimir variables de entorno crudas en un script y capturar la salida en el log del LLM. | Fuga de secretos (API Keys, JWTs) al LLM host externo. | Sanitización profunda (Scrubbing) de Stdout previo al Logger |
| ADV-052 | Robo de Foco (Window Stealing) | UI Mac Control | Un script asíncrono lanza una ventana de popup invisible en macOS que se roba el foco de teclado del Operador. | Intercepción de claves, interrupción del flujo biológico. | Ejecución en contextos sin cabeza (Headless) o monitoreo estricto de PIDs visuales |
| ADV-053 | Inyección de Falso Positivo Lint | Pre-commit Hooks | Añadir reglas de Linter custom al repositorio que marcan todo código nuevo de MOSKV-1 como erróneo siempre. | El sistema nunca puede comitear la solución, estancamiento total. | Reglas Linter Bloqueadas, inmutables desde CORTEX Core (`pyproject.toml` sellado) |
| ADV-054 | Desborde de JSON por Profundidad | `run_command` JSON | Pedirle a una API externa que retorne un JSON anidado 50,000 niveles. | RecursionError al intentar hacer `json.loads()`, muerte del script. | Límite de Max Depth en validadores Pydantic / ORJSON limit |
| ADV-055 | Secuestro de Configuración `.git/config` | Tool `run_command` | Cambiar el `remote.origin.url` para que el Git Sentinel suba el código C5-REAL a un repositorio atacante privado. | Exfiltración silenciosa del codebase entero. | Hash Integrity Check de los archivos de configuración `.git/config` |
| ADV-056 | Desacople de Logs (Log Splitting) | Consola Estándar | Inyectar retornos de carro (`\r`) y Backspaces (`\b`) para borrar u oscurecer logs de advertencia en TTY en vivo. | Ceguera temporal del Operador Biológico sobre lo que ocurre. | Filtrado estricto de ANSI Escape Sequences en la ingesta |
| ADV-057 | Inducción Térmica Artificial | Operaciones Computacionales | Hacer operaciones de tensor matricial repetitivas de forma continua para saturar GPU/CPU y freír térmicamente la cuota. | Ralentización de CORTEX, caída térmica. | Límite de Cuota Estricto de ciclos o Auto-Pause T>90ºC |
| ADV-058 | Aislamiento Simulado (Mock Fakeout) | Tests Unitarios | Sobreescribir las herramientas de Mock (`unittest.mock`) para que aprueben sin validar. | Aprobación de pruebas críticas que realmente rompen el sistema. | Ejecución pura de Tests de Integración End-to-End sin Mocks globales |
| ADV-059 | Manipulación de Prompt de Sistema Base | Configuración `.agents/` | Escribir un archivo oculto (por ej. una regla en `.agents/AGENTS.md`) que invierte el comportamiento C5. | Pérdida total de Soberanía del Motor (Jailbreak masivo permanente). | Firma de Hash del Directorio Config (Tamper Evident) |
| ADV-060 | Inyección de Grafo Recursiva | Memoria Neo4j / NetworkX | Insertar nodos masivos conectados a un Nodo Central ("Hub") rompiendo la búsqueda de vecinos más cercanos. | Caída del motor de grafos al hacer un "Expand All". | Paginación en Traverse de Nodos (Límite dinámico de Depth/Neighbors) |
