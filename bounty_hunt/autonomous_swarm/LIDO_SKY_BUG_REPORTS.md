**Severity:** High (Double Spend / Frozen Fund Extraction)
**Impact:** If the Lido oracle network undergoes a 12-second block partition (common in MEV-boost relays or Dencun hardfork latency spikes), an attacker can flood `requestWithdrawals` identical permutations. **Proof of Concept:**
See `poc_lido/LidoWithdrawalDoubleSpendPoC.sol`.

**Remediation (Ouroboros Logic Synthesis):**
