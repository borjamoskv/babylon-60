# CEP-005: Security & Threat Model

**Status:** Draft
**Layer:** Security
**Authors:** BABYLON-60 Architecture Group

## 1. Scope
This document specifies the official threat model for the BABYLON-60 microkernel, detailing attack vectors, cryptographic guarantees, and mitigation strategies.

## 2. Terminology
The key words "MUST", "MUST NOT", "REQUIRED", "SHALL", "SHALL NOT", "SHOULD", "SHOULD NOT", "RECOMMENDED", "MAY", and "OPTIONAL" in this document are to be interpreted as described in RFC 2119.
