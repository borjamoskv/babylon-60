# CEP-006: Conformance Test Suite

**Status:** Draft
**Layer:** Testing
**Authors:** BABYLON-60 Architecture Group

## 1. Scope
This document specifies the official conformance test suite that any independent BABYLON-60 Microkernel implementation MUST pass to be certified as fully compliant.

## 2. Terminology
The key words "MUST", "MUST NOT", "REQUIRED", "SHALL", "SHALL NOT", "SHOULD", "SHOULD NOT", "RECOMMENDED", "MAY", and "OPTIONAL" in this document are to be interpreted as described in RFC 2119.
